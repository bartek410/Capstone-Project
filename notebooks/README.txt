Genre Classification
This project aims to use machine learning to classify songs into their genre. 

Documents in this project folder (excluding this one) include: 
- 5 Jupyter Notebooks 
- A data folder containing 7 csv files 
- PDF of Presentation Slides
- READMESummary.md

Order of Notebooks: 
- 1A - Spotify Dataset 2 Cleaning & EDA
- 1B - Spotify Dataset 1 Cleaning and EDA (used for cleaning a dataset that was used in notebook 3)
- 2 - Spotify Consolidating Genres
- 3 - Modeling_Spotify_Datasets 
- 4 - Spotify_Genre_Clustering
